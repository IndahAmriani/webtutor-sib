Panduan Penggunaan SIPALING

Sistem Informasi Pengujian Laboratorium

SIPALING adalah aplikasi berbasis web yang digunakan untuk mengelola proses pengujian laboratorium, mulai dari data pelanggan, perhitungan biaya pengujian, hingga pembuatan surat biaya secara otomatis.

ℹ️ Tentang Aplikasi

Aplikasi SIPALING dibangun menggunakan Laravel untuk backend dan Vue.js untuk frontend. Sistem ini membedakan hak akses antara Admin dan Pegawai, di mana Admin memiliki akses penuh untuk mengelola data, sedangkan Pegawai dapat melihat data dan mengunduh dokumen.

Buat Surat Biaya

Halaman inti untuk menerbitkan surat biaya baru (Hanya untuk Admin)

ℹ️ Akses Terbatas

Halaman ini hanya dapat diakses oleh pengguna dengan role Admin.

📋 Tahapan Pembuatan Surat Biaya

1

Isi Informasi Dasar

Mengisi data dasar surat biaya:

  • Tanggal Surat: Tanggal penerbitan surat
  • Pilih Pelanggan: Pilih dari data pelanggan yang tersedia
  • Pilih Tanda Tangan: Pilih pejabat penanda tangan
  • Pilih Nomor Rekening: Pilih rekening tujuan pembayaran
2

Tambah Biaya Pengujian Sampel

Menambahkan biaya pengujian sampel:

  • Pilih Matriks pengujian (misal: Air Bersih, Air Laut, Udara Ambient)
  • Pilih satu atau lebih Parameter pengujian pada matriks tersebut
  • Isi Volume untuk setiap parameter
  • Harga parameter akan otomatis terisi
  • Sistem menghitung sub total per matriks secara otomatis

💡 Fitur Subkon

Centang checkbox "Subkon" jika sampel disubkontrakkan. Ini akan menambah keterangan "Tanda (*) Sampel yang disubkon" pada catatan.

3

Isi Biaya Sampling (Operasional)

Mengisi biaya operasional secara manual:

  • Uang Harian PPC (OH): Harga satuan × volume
  • Transport PPC (OT): Harga satuan × volume
  • Akomodasi PPC (OK): Harga satuan × volume
  • Penyeberangan Mobil Sampling (PP): Harga satuan × volume
  • BBM Kendaraan Sampling: Harga satuan × volume
  • BBM Genset: Harga satuan × volume

Sistem otomatis mengalikan harga satuan dengan volume untuk mendapatkan jumlah biaya.

4

Isi Catatan dan Status Pembayaran

  • Catatan: Isi catatan tambahan (opsional)
  • Status Pembayaran: Pilih status (Belum Bayar, Bayar Sebagian, atau Lunas)

💡 Status Bayar Sebagian

Jika memilih "Bayar Sebagian", sistem akan menampilkan kolom jumlah yang telah dibayar dan otomatis menghitung sisa pembayaran.

5

Pratinjau (Preview) Surat Biaya

Sistem menampilkan pratinjau surat biaya secara real-time di sisi kanan halaman. Anda dapat melihat tampilan dokumen sambil mengisi data.

ℹ️ Nomor Surat Otomatis

Nomor surat biaya dibuat secara otomatis (auto-generated) mengikuti format tanggal dan urutan penerbitan pada hari tersebut.

6

Simpan & Export DOCX

Setelah seluruh data terisi dengan benar, klik tombol "Simpan & Export DOCX" untuk:

  • Menyimpan data surat biaya ke database
  • Mengunduh dokumen dalam format Microsoft Word (.docx)

Matriks & Parameter

Kelola data matriks pengujian dan parameter (Hanya untuk Admin)

ℹ️ Akses Terbatas

Halaman ini hanya dapat diakses oleh pengguna dengan role Admin.

⚠️ Penting

Data pada halaman ini menjadi acuan utama dalam perhitungan biaya pengujian sampel pada halaman Buat Surat Biaya.

📋 Bagian 1: Data Matriks

1

Contoh Matriks Pengujian

  • Air Bersih
  • Air Permukaan
  • Air Laut
  • Udara Ambient
  • Lingkungan Kerja Fisika
  • Dan lain-lain
2

Kelola Data Matriks

Admin dapat menambah, mengubah , atau menghapus data matriks sesuai kebutuhan operasional laboratorium.

📋 Bagian 2: Data Parameter

3

Informasi Parameter

Setiap matriks memiliki parameter pengujian dengan informasi:

  • Nama Parameter: Jenis parameter yang diuji
  • Metode: Metode analisis yang digunakan
  • Harga: Tarif pengujian per parameter
4

Kelola Data Parameter

Admin dapat menambah, mengubah , atau menghapus parameter pada matriks tertentu.

💡 Tujuan Pengelolaan Data

Pengelolaan data yang terstruktur menjaga kesesuaian antara parameter pengujian, metode analisis, serta tarif yang digunakan dalam perhitungan biaya.

User Sistem

Kelola akun pengguna sistem (Hanya untuk Admin)

ℹ️ Akses Terbatas

Halaman ini hanya dapat diakses oleh pengguna dengan role Admin.

📋 Kelola Akun Pengguna

1

Tambah User Baru

Klik tombol tambah untuk membuat akun pengguna baru:

  • Username
  • Password
  • Role (Admin atau Pegawai)

Klik tombol edit untuk mengubah data pengguna yang ada.

Klik tombol hapus untuk menghapus akun pengguna. Dialog konfirmasi akan muncul sebelum penghapusan.

Sistem menerapkan ketentuan keamanan password:

  • Minimal 8 karakter
  • Harus mengandung huruf besar
  • Harus mengandung huruf kecil
  • Harus mengandung angka
  • Harus mengandung simbol

Logout

Cara keluar dari sistem dengan aman

📋 Langkah-langkah Logout

1

Klik Tombol Logout

Temukan tombol logout pada menu navigasi atau pojok kanan atas halaman.

2

Konfirmasi Logout

Sistem akan menampilkan dialog konfirmasi. Klik "Ya" untuk melanjutkan logout.

3

Redirect ke Halaman Login

Setelah logout, Anda akan diarahkan kembali ke halaman login.

💡 Tips Keamanan

Selalu lakukan logout setelah selesai menggunakan aplikasi, terutama jika menggunakan komputer bersama atau di tempat umum.
